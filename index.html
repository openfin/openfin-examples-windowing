The OpenFin Window API

OpenFin’s Windowing API offers granular control over the creation, positioning, minimization, maximization and a host of other properties, across multiple monitors, allowing you create rich desktop applications. The full API may be found here.

Draggable Regions

This example shows how to implement draggable regions on frameless windows and how to avoid some of the pitfalls which may arise from the draggable region’s prevention of standard mouse events.

Check it out here.

Closing Events and Child Windows

OpenFin allows the creation of two types of windows–child windows and applications. This example demonstrates the differences between the two, and how closing of parent and child windows may be controlled.

Snapping and Docking

If you wish to implement window docking, we have produced a library which may be used as-is, or modified for specific behavior here.

An annotated implementation of the Snap and Dock library may be found here.

Tear-Out Windows

OpenFin allows elements to be ‘torn out’ of one window and placed on the desktop or dropped into another window. This example shows how the effect may be achieved.

Alpha Transparency

NB: This is a Beta feature. It is possible to set a specific color to be an alpha channel, allowing transparent windows.
